# Java implementation of GoF patterns

### Available patterns:
* Behavioral
  * Command
  * Iterator (Custom Iterator, Java Iterator)
  * Observer
  * Strategy
  * Chain of Responsibility
  * Interpreter
  * State
  * Template method
  * Mediator
* Creational
  * Builder
  * Factory (Factory Method, Abstract Factory)
  * Singleton
  * Prototype
* Sructural
  * Adapter (Object Adapter, Class Adapter)
  * Decorator
  * Facade
  * Proxy
  * Flyweight
  * Composite
  * Bridge
