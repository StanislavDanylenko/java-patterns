# Java implementation of GoF patterns

### Available patterns:
* Behavioral
  * Command
  * Iterator (Custom Iterator, Java Iterator)
  * Observer
  * Strategy
* Creational
  * Builder
  * Factory (Factory Method, Abstract Factory)
  * Singleton
  * Prototype
* Sructural
  * Adapter (Object Adapter, Class Adapter)
  * Decorator
  * Facade
  * Proxy
  * Flyweight
